## Install

Clone the GitHub Repo locally

```
git clone https://github.com/DaanKetelaars/front-end-data
```

Install the Packages

```
npm install
```

Start the Server

```
npm start
```

Make sure to tag d3.js in your html file
```
<script src="https://d3js.org/d3.v7.min.js"></script>
```
